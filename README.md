## Files

### `src/index.html`

This file is the main HTML file for the website. It contains the structure and content of the salon's website. It includes sections for services, stylists, gallery, and contact information.

### `src/styles/main.css`

This file contains the CSS styles for the website. It is used to customize the appearance and layout of the HTML elements. It includes styles for headings, paragraphs, buttons, navigation menu, and other elements.

### `src/scripts/main.js`

This file contains the JavaScript code for the website. It is used to add interactivity and dynamic behavior to the website. It includes functions for form validation, image sliders, and booking appointment functionality.

### `src/images`

This directory is used to store the images used in the website, such as salon logos, stylist photos, or hairstyle examples. It includes subdirectories for different categories of images.

### `package.json`

This file is the configuration file for npm. It lists the dependencies and scripts for the project. It includes packages for building, testing, and running the website.

### `README.md`

This file contains the documentation for the project. It provides information about the project's purpose, setup instructions, and any other relevant details.

## Setup

1. Clone the repository.
2. Install the dependencies using `npm install`.
3. Open `src/index.html` in a web browser to view the website.

## Usage

- Browse the website to learn about the salon's services and stylists.
- Use the contact information to get in touch with the salon.
- Book an appointment using the online booking form.
